JavaScript之Math方法

JavaScript Math 对象

语法 返回值 描述
sin(x)、asin(x)、cos(x)、acos(x)、tan(x)、atan(x)
abs(x)、exp(x)、log(x)、pow(x,y)、sqrt(x) 绝对值/E 的 x 次方/自然对数/x 的 y 次幂/平方根
ceil(x)、floor(x)
max(x,y,z,…,n)、min(x,y,z,…,n)
random()、round(x)
atan2(y,x) 返回从 x 轴到点 (x,y) 的角度（介于 -PI/2 与 PI/2 弧度之间）

详细用法

方法 返回值 描述
abs(x) NaN/0/number
acos(x) NaN/0~PI
asin(x) -PI/2 到 PI/2I
atan(x) -PI/2 到 PI/2
atan2(y,x) -PI/2 到 PI/2
ceil(x) 大于等于 x
cos(x) -1.0 到 1.0
exp(x) number
floor(x) 小于等于 x
log(x) x 的自然对数/NaN/ -Infinity
max(x,y,z,…,n) 参数中最大值/-Infinity/NaN
min(x,y,z,…,n)
pow(x,y) number
random() 0.0 ~ 1.0（不包含）
round(x) 最接近的整数
sin(x) -1.0 到 1.0
sqrt(x) Number/NaN
tan(x) Number
