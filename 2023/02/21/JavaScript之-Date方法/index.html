JavaScript之 Date方法

JavaScript 之 Date 对象

语法 返回值 描述
getDate()、getUTCDate()、setDate()、setUTCDate() 1-31
getDay()、getUTCDay() 0-6
getFullYear()、getUTCFullYear()、setFullYear()、setUTCFullYear() 四位数字
getHours()、getUTCHours()、getHours()、getUTCHours() 0-23
getMilliseconds()、getUTCMilliseconds()、setMilliseconds()、setUTCMilliseconds() 0-999
getMinutes()、getUTCMinutes()、setMinutes()、setUTCMinutes() 0-59
getSeconds()、getUTCSeconds()、setSeconds()、setUTCSeconds() 0-59
getTime()、UTC()、setTime()、parse() number
toString()、toDateString()、toTimeString() String 把 Date 对象/日期部分/时间部分转换为字符串
toLocaleString()、toLocaleDateString()、toLocaleTimeString() String 根据本地时间把 Date 对象转换为字符（同 toString）
toISOString() String 使用 ISO 标准将 Date 对象转换为字符串
toJSON() String 将 Date 对象转换为字符串，并格式化为 JSON 数据格式
toUTCString() String 根据世界时 (UTC) 把 Date 对象转换为字符串
valueOf() String 返回 Date 对象的原始值

UTC: 世界时
parse(): 返回 1970 年 1 月 1 日午夜到指定日期（字符串）的毫秒数
