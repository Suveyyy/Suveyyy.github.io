JavaScript之Array方法

JavaScript Array 对象

方法 返回值 语法
concat（） 新数组
copyWithin() 改变原数组 从数组的指定位置拷贝元素到数组的另一个指定位置中
entries() 一个数组的迭代对象 把数组变成对象
every() true/false
fill() 改变原数组
filter() 新数组
find() 数组元素/undefined
findIndex() number/-1
forEach() 返回值
from() 新数组
includes() true/false
indexOf() number/-1 查找数组首次出现的位置
isArray() true/false Array.isArray(obj)
join() string
keys() 一个数组迭代对象
lastIndexOf() number/-1
map() 新数组
pop() 改变原数组
push() 改变原数组
reduce() number
reduceRight() number 功能和  reduce()  功能是一样的，不同的是 reduceRight() 从数组的末尾向前将数组中的数组项做累加
reverse() 改变原数组
shift() 改变原数组
slice() 新数组
some() true/false
sort() 改变原数组
splice() 改变原数组
toString() string
unshift() 改变原数组
valueOf() object 把数组变成对象

删除数组指定元素：
splice()会改变原数组
filter()不会改变原数组
